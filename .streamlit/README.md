### Streamlit Secrets Setup

This project requires a **GROQ API Key** for the AI chatbot functionality.

Create a file named `secrets.toml` inside the `.streamlit` folder and add the following:

```
GROQ_API_KEY="your_groq_api_key_here"
```

Replace `your_groq_api_key_here` with your actual GROQ API key.

Note: The `secrets.toml` file is not included in this repository for security reasons.
